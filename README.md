# Task Tracker

## Built using:
- Spring Boot
- React JS and Tailwind
- PostgreSQL

## Features
- Implemented Spring Security with oAuth2.0
- Dynamically updating page using react hooks

## Project Structure

### Client
- Pages
- Components
- Context
- Assets
### Server
- Controllers
- Repositories
- Services
- Models
- DTOs
- Config
